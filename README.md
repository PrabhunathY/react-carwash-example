React JS Car Wash web application is a SaaS-based web application POC example.

### Available Scripts

In the project directory, you can run:
#### with `yarn` command
#### `yarn install`
#### `yarn start`

#### with `npm` command
#### `npm install`
#### `npm start`
Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
